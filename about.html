About Runestone Interactive

The Runestone Interactive project aims to do two things. 1) Provide open source authoring tools for creating and hosting quality interactive learning resources. 2) Host custom versions of textbooks for colleges, universities, and high schools around the world.

Started in 2011, the first versions of Runestone textbooks went public in June 2012, we have since grown to over 5000 students a day, and are used in over 100 institutions around the world.

Related Projects

Runestone Interactive would not be what it is without the components from other open source projects

  • Skulpt.org -- A Javascript implementation of the Python programming language. This allows us to run Python examples right in the browser!
  • Online Python Tutor-- Philip Guo's excellent project for visualizing Python Programs
  • Sphinx-- Python Documentation Generator
  • Parson's Problems-- An open source library developed by Ville Karavirta
  • Blockly-- An open source project for a blocks based language
